# Z80DevBoard
An advanced development board for the Z80 (Specific model: Z8400).

Gianluca Rainis (gianluca-rainis on GitHub)

## Developing tools
- KiCad

## Components
- Z8400
- RP2040
- HM62256BLP (RAM)
- W25Q32JVSS (Flash)
- Crystal oscillator
- AMS1117-3.3 (5V to 3v3)
- 74HC595 (Serial to 8 bit)
- 74HC165 (8 bit to Serial)
- SN74LVC245APW (Bus to handle the voltage differences)
- USB C 2.0 port
- Buttons
- Leds
- Resistors 0 ohm
- Resistors 220 ohm
- Resistors 5k ohm
- Resistors 10k ohm
- Capacitors 0.1uF
- Capacitors 1uF
- Capacitors 10uF
- Fuse 500mA
