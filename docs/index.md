## About This Document
This documentation is designed for use in educational kits. Each chapter is self-contained and can be read independently, though following the order is recommended for beginners.

The document is structured in two layers:
- **Introductory sections** at the start of each chapter, suitable for anyone with basic electronics curiosity
- **Technical deep-dives** for those who want to understand every signal, register, and timing detail
